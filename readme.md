# Trabajo Práctico - Visión por Computadora II

Este repositorio contiene el desarrollo del Trabajo Práctico de la materia **Visión por Computadora II**. El objetivo es realizar una tarea de **clasificación de residuos** utilizando el dataset **TrashNet**.

## Descripción del Problema
El objetivo es entrenar un modelo de clasificación que permita identificar distintos tipos de residuos (plástico, vidrio, papel, orgánicos) a partir de imágenes, contribuyendo a la optimización de procesos de reciclaje.

Dataset utilizado:
- **TrashNet Dataset**: https://www.kaggle.com/datasets/feyzazkefe/trashnet

## Contenidos del Trabajo
- Análisis exploratorio del dataset
  * Analisis de distribución de colores
  * Análisis de distribucioón de etiquetas
  * Dimensin de imágenes y relación de aspecto
  * Calidad de imágenes
- Entrenamiento de modelos de clasificación
- Comparación de arquitecturas y métricas
- Data augmentation y/o transfer learning según sea necesario
